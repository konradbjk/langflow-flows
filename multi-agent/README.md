# Langflow Flows

This folder contains example Langflow flows as JSON files that you can import into Langflow.

**How To Use**
1. Open Langflow.
2. Import a JSON flow from this folder.
3. Update provider credentials (LLM, embeddings, vector store, search tools) as needed.

**Flows**
- `DeepResearch-Langflow-Blog-Example.json`
- `DeepResearch-Tavily.json`
- `Guardrails-RAG-Pipeline.json`
- `Multi-Query-Retriever.json`
- `Naive-RAG-Qdrant.json`
- `Query-Rewriting.json`
